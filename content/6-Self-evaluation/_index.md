During my internship at **Amazon Web Services Viet Nam** from **17/04/2026** to **10/07/2026**, I had the opportunity to learn, train, and apply my accumulated professional knowledge into practical work, while gradually improving programming skills and team management capabilities necessary for future development.

Specifically, I participated in building the **Cloud Nexus** project – a network simulation and security analysis platform that allows users to design network diagrams and use AI (Google Gemini) to detect vulnerabilities, simulate attack paths, and recommend defensive measures. In this project, I simultaneously held two roles: **Leader** and **Developer**.

---

## My Role as Developer

I directly participated in designing the system architecture and programming the deployment of core modules on the AWS platform. I was responsible for building processing flows from user frontend access, configuring authentication via Cognito, setting up API Gateway, to programming complex asynchronous processing flows using SQS, EventBridge Pipes, Step Functions, and Lambda. Additionally, I integrated DynamoDB databases, S3, called AI APIs to analyze results, and set up alert and monitoring systems via SNS and CloudWatch.

---

## My Role as Team Leader

I was responsible for planning, task distribution, and managing the overall progress of the team. I regularly performed code reviews, assisted in troubleshooting difficult technical issues, and ensured that serverless components developed by other members could integrate smoothly into a complete system.

---

Through these two roles, I not only improved my programming skills and deeply understood how to optimize an AWS system in practice, but also learned how to coordinate projects, communicate technical ideas, and bond a team to achieve common goals.

Throughout the internship, I always strived to complete assigned tasks well, followed company regulations, and proactively communicated with Mentor/Admin and colleagues to improve work efficiency and quality.
